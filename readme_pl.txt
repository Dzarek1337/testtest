m-TVGuide

Jest to wtyczka dla Kodi z obsługą elektronicznego przewodnika telewizyjnego (EPG),
która umożliwia strumieniowe przesyłanie treści udostępnionych przez 
koncesjonowanego nadawę telewizji.

Lista dostępnych nadawców telewizji

C More - C More Entertainment
Cyfrowy Polsat GO - Cyfrowy Polsat S.A.
Ipla - Cyfrowy Polsat S.A.
nc + GO - ITI Neovision S.A.
PlayerPL - TVN S.A.
Telia Play - Telia Company AB
WP Pilot - Wirtualna Polska Media S.A.


Zrzeczenie

Wtyczka nie jest oficjalna koncesjonowanego dostawy, oraz nie jest przez niego zatwierdzona 
i wspierana. Wszelkie użyte znaki handlowe należą do firm i organizacji podmiotu koncesjonowanego.

Logowanie do stron internetowych odbywa się za pośrednictwem usług operatora telewizji, 
jest udostępniane przez podmiot Application Programming Interface (API).

Strumienie chronione DRM

Niektóre treści są chronione DRM i wymagają modułu deszyfrującego Widevine CDM do ich odtwarzania. 
Podczas próby uruchomienia takiej treści zostaniesz poproszony o instalację dodatkowego modułu jeśli 
strumień nie posiada zainstalowanego pliku binarnego.

Większość urządzeń z Androidem ma wbudowaną obsługę Widevine DRM i nie wymaga dodatkowych instalacji 
plików binarnych. Możesz to sprawdzić za pomocą odpowiedniej aplikach udostępnionej w sklepie Play.
